什么是ACM-ICPC国际大学生程序设计竞赛

ACM国际大学生程序设计竞赛（英文全称：ACM International Collegiate Programming Contest（简称ACM-ICPC或ICPC））是由国际计算机协会（ACM）主办的，一项旨在展示大学生创新能力、团队精神和在压力下编写程序、分析和解决问题能力的年度竞赛。经过近40年的发展，ACM国际大学生程序设计竞赛已经发展成为全球最具影响力的大学生程序设计竞赛。

总而言之，ICPC是最公平、最盛大、难度最高的国际大学生程序设计竞赛

简要规则

ACM-ICPC以团队的形式代表各学校参赛，每队由3名队员组成。

比赛期间，每队使用1台计算机需要在5个小时内使用C、C++或Java中的一种编写程序解决8到13个问题。程序完成之后提交裁判运行，运行的结果会判定为:

  • AC（正确）
  • WA（错误）
  • TLE（超时）
  • MLE（超出内存限制）
  • RE（运行错误）
  • PE（格式错误）

中的一种并及时通知参赛队。每队在正确完成一题后，组织者将在其位置上升起一只代表该题颜色的气球。最后的获胜者为正确解答题目最多且总用时最少的队伍。每道试题用时将从竞赛开始到试题解答被判定为正确为止，其间每一次提交运行结果被判错误的话将被加罚20分钟时间，未正确解答的试题不记时。

与其它计算机程序竞赛（例如国际信息学奥林匹克，IOI）相比，ACM-ICPC的特点在于其为3人团队作战且题量大，每队需要5小时内完成8道题目，甚至更多。除此之外，一支队伍有3名队员却只有1台计算机，使得时间显得更为紧张。因此除了扎实的编程能力，良好的团队协作和心理素质同样是获胜的关键。

考察范围

  1. 使用C/C++或JAVA编写程序的能力
  2. 算法、数据结构、数学能力等

主要比赛

  1. CCCC中国大学生团体程序设计天梯赛 每年三月底
  2. 陕西省ACM程序设计竞赛（全国邀请赛） 每年五月左右
  3. ICPC亚洲区域赛——网络预选赛 八到十月
  4. ICPC亚洲区域赛 十到十二月
  5. ICPC World Final 世界总决赛 每年四月
  6. 蓝桥杯
  7. 各校校赛等

竞赛队近几年的获奖成果

  1. 2015年在陕西省第三届大学生程序设计竞赛取得2个银奖、2个铜奖和一个最佳女队奖

  2. 2016年在ACM-ICPC全国邀请赛（西安站）获得一银四铜，在ACM-ICPC区域赛（大连站）和China Final中获得铜牌

  3. 2017年在ACM-ICPC全国邀请赛（西安站）获得一银一铜，并且获得一枚省级金牌

  4. 2018年在中国高校计算机大赛-团体程序设计天梯赛（以下简称CCCC比赛）中获得二等奖、三等奖，ACM-ICPC全国邀请赛（西安站）获得铜牌，ACM-ICPC区域赛（青岛站）获得一枚铜牌，ACM-ICPC亚洲区决赛中获得铜牌

  5. 2019年CCCC比赛中获得三等奖，ACM-ICPC全国邀请赛获得铜牌，第七届陕西省ACM-ICPC比赛中获得2枚铜牌。

福利与缺点

福利

  1. 公费旅游(亚洲区域赛每年全国多地举办，如2019年举办地为 辽宁沈阳 江苏徐州 江苏南京 中国上海 中国香港 甘肃银川 江西南昌 陕西西安)，学校将为参赛队伍提供报销。
  2. 增强代码能力！Coding能力！
  3. 为今后工作笔试、面试助力（绝大多数笔试面试题都和ACM题目相近，且没有ACM难）
  4. 通吃算法与数据结构
  5. 综合测评up！up！

缺点

  1. 难以通过ACM保研
  2. 很多同学刚开始觉得很感兴趣，但是随着学习难度加大很难坚持下去
