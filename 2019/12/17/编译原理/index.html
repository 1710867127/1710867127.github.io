绪论

什么是编译

编译程序的结构

词法分析器：又称扫描器，输入源程序，进行词法分析，输出单词符号

语法分析器：又称分析器，对单词符号进行语法分析（根据语法规则进行推导或归约），识别出各类语法单位，最终判断输入串是否构成语法上正确的”程序”

语义分析与中间代码产生器：按照语义规则对语法分析器归约出的语法单位进行语义分析并把它们翻译成一定形式的中间代码

优化器：对中间代码进行优化处理

目标代码生成器：把中间代码翻译成目标程序

基本概念

$\Sigma$ 是一个有穷字母表 ，它的每个元素成为一个符号。$\Sigma$ 上的一个符号串是指由 $\Sigma$ 中的符号所构成的一个有穷序列。不包含任何符号的序列成为空字，记为 $\varepsilon$ 。用 $\Sigma^*$ 表示 $\Sigma$ 上的所有符号串的全体，空字 $\varepsilon$ 也包括其中。例如，若 $\Sigma=\{a,b\}$ ，则 $\Sigma^* = \{\varepsilon,a,b,aa,ab,ba,bb,aaa,…\}$ 。 $\phi$ 表示不含任何元素的空集。要注意 $\varepsilon$ 、 $\{\}$ 、 $\{\varepsilon\}$ 的区别。

上下文无关文法

文法是描述语言的语法结构的形式规则（即语法规则）。一个上下文无关文法G包括四个部分：一组终结符号、一组非终结符号、一个开始符号、以及一组产生式。

终结符号是组成语言的基本符号。

非终结符号用来代表语法范畴。

开始符号是一个特殊的非终结符号。

产生式是定义语法范畴的一种书写规则。一个产生式的形式是

$A \rightarrow \alpha $

为了书写方便，若干个左部相同的产生式可以合并为一个，缩写成

$A \rightarrow \alpha_1 | \alpha_2 | … | \alpha_n$

每一个 $\alpha_i$ 也称为P的一个候选式

假定G是一个文法，S是它的开始符号。如果 $S \overset{*}{\rightarrow} \alpha $ ,则称 $\alpha$ 是一个句型。进包含终结符号的句型是一个句子。 文法G所产生的句子的全体是一个语言，将它记为L(G).

最左推导：任何一步 $\alpha \rightarrow \beta$ 都是对 $\alpha$ 中的最左非终结符进行替换的。同理我们可以定义最右推导。

语法分析树与二义性

基本概念

用一张图来表示一个句型的推导，这种表示称为语法分析树。根节点由开始符号所标记。

如果一个文法存在某个句子对应两棵不同的语法树，则称这个文法是二义的。也就是说，若一个文法中存在某个句子，它有两个不同的最左(右)推导，则这个文法是二义的。

第一章 引论

1、宿主机、目标机概念，何谓编译程序？编译程序的输入输出是什么？

2、编译程序的“前端”、“后端”主要由哪些部分组成？它们与目标机器有何关系？

3、编译程序各个阶段的输入输出、所依循的规则及描述该规则的工具。结合图1.1。

answer
1、运行编译程序的计算机称宿主机，运行编译程序所产生的目标代码的计算机称为目标机。

把某种语言程序转换成另一种语言程序，如果源语言是“高级语言”，而目标程序是“低级语言”，这样的一个翻译程序就称为编译程序。

输入：源程序 输出：目标程序
2、前端主要由与源语言有关但与目标机器无关的部分组成，通常包括词法分析、语法分析、语义分析与中间代码生成。后端包括编译程序中与目标机有关的部分，如与目标机有关的代码优化和目标代码生成等。

可以取编译程序的前端，改写后端以生成不同目标机上的相同语言的编译程序。这样就可以实现编译程序的目标机的改变。也可以讲几种源语言编译成相同的中间语言，然后为不同的前端配上相同的后端，这样就可以为一台机器生成不同语言的编译程序。
3、第一阶段，词法分析。 输入源程序，输出单词符号。 在词法分析阶段所依循的是语言的词法规则(构词规则)。描述词法规则的有效工具是正规式和有限自动机。

第二阶段，语法分析。 在词法分析的基础上，根据语言的语法规则，把单词符号分解成各类语法单位。(输入：单词符号，输出：各类语法范畴) 语法分析所依循的是语言的语法规则，语法规则通常用上下文无关文法描述。

第三阶段，语义分析与中间代码产生 分析语法的含义，并初步翻译。一方面对每种语法范畴进行静态语义检查，另一方面进行中间代码的翻译。(输入：语法范畴，输出：中间代码) 所依循的是语言的语义规则。
第四阶段，优化。 对前端产生的中间代码进行加工变换，主要有公共表达式的提取、循环优化、删除无用代码等。(输入：中间代码，输出：高效的中间代码) 所依循规则是程序的等价变换规则。

第五阶段，目标代码生成 把中间代码变化成待定机器上的低级语言代码。(输入：中间代码，输出：目标代码) 工作有赖于硬件系统结构和机器指令含义。

answer
5、最左推导：任何一步 $\alpha \rightarrow \beta $ 都是对 $\alpha$ 中的最左非终结符进行替换的。

最右推导：同理

语法分析树：用一张图来表示一个句型的推导，这种表示称为语法分析树。
6、乔姆斯基把文法分为四种
0型文法也叫短语文法，任何0型文法都是递归可枚举的；反之，递归可枚举集必定是一个0型语言。

1型文法也叫上下有关文法。对非终结符替换时务必考虑上下文，并且一般不允许替换成空串 $\varepsilon$ 。G的任何产生式 $\alpha \rightarrow \beta $ 均满足 $ \		 \alpha \ \leq \ \beta \ $ .(其中 $\ \alpha \ $ 和 $\ \beta \ $ 分别为 $\alpha $ 和 $\beta $ 的长度)；仅仅 $S \rightarrow \varepsilon $ 例外，但S不得出现在任何产生式右部。
